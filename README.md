# demo c++ projects

## server

### build

```
cd demo/server
module clean build
```

### run

```
./build/server <args>
```

### help

```
./build/server --help
```

## argparse

### build

```
cd demo/argparse
C++ -std=c++20 argparse.cpp -o argparse
```

### run

```
./argparse <pass it some arguments>
```
